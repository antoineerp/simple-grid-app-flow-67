    Interprétation des résultats

    Si les tests ci-dessus sont réussis, votre serveur est correctement configuré pour servir des modules ES6. Voici les éléments importants à vérifier:

    • Le type MIME du fichier JavaScript doit être application/javascript ou text/javascript
    • Le module ES6 doit être chargé avec succès via l'instruction import
    • Votre navigateur doit supporter les modules ES6 (tous les navigateurs modernes le font)
