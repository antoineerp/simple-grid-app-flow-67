Comment procéder

Accédez à phpMyAdmin via votre compte Infomaniak

Connectez-vous à votre espace client Infomaniak et accédez à la section "Hébergement Web" puis "Bases de données MySQL".

Exécutez le script SQL de diagnostic

Utilisez le script SQL fourni dans le fichier "diagnostics-infomaniak.sql" pour tester votre connexion.

-- Copier depuis diagnostics-infomaniak.sql
SELECT VERSION() AS mysql_version, DATABASE() AS current_database, USER() AS current_user;
SHOW TABLES;

Vérifiez les résultats

Si vous voyez les informations de version MySQL et la liste des tables, la connexion est réussie.

Alternative: MySQL Workbench

Si vous avez accès à MySQL Workbench ou un autre client MySQL, vous pouvez utiliser les informations de connexion ci-dessus pour établir une connexion directe.

Que faire en cas de problème

  1. Vérifiez que les informations de connexion sont correctes
  2. Assurez-vous que votre adresse IP est autorisée à se connecter (vérifiez les restrictions dans votre panneau Infomaniak)
  3. Vérifiez que la base de données existe et que l'utilisateur a les droits suffisants
  4. Contactez le support Infomaniak si les problèmes persistent
